# LLM-Powered Agentic Trading System

A multi-tool LLM agent that retrieves market data and performs technical 
analysis using function calling. The agent evaluates indicators to generate 
trade rationales, tested via simulated backtesting.

## Key Features
- Multi-tool agent architecture using LangChain Agents
- Function calling for real-time technical analysis:
  - RSI (Relative Strength Index)
  - MACD (Moving Average Convergence Divergence)
  - Bollinger Bands
- Market data retrieval pipeline
- Evaluation loop for agent-generated trade rationales

## Key Results
- 12% simulated portfolio growth in backtesting

## Tech Stack
Python, LangChain Agents, GPT-4, Pandas
